## 📖 Sobre o Projeto

O **Cantina Bem-Estar** é uma solução web completa e robusta para a administração de cantinas escolares. O projeto nasceu como nosso TCC do curso técnico e foi concebido para otimizar a gestão, automatizar processos e melhorar a experiência tanto dos administradores quanto dos clientes (alunos e funcionários).

A plataforma centraliza o controle de estoque, o registro de vendas e a geração de relatórios financeiros, fornecendo aos gestores uma visão clara e detalhada do negócio. Para os clientes, oferece uma interface amigável para interação com a cantina.

## ✨ Principais Funcionalidades

-   **👥 Interface do Cliente:** Área dedicada aos clientes, com histórico de compras e um ranking de fidelidade.
-   **📊 Relatórios Financeiros:** Geração de relatórios detalhados de faturamento, lucro e produtos mais vendidos.
-   **🧑‍💻 Dashboard Administrativo:** Painel central com gráficos e indicadores-chave de desempenho (KPIs) para tomada de decisão estratégica.
-   **🏆 Ranking de Clientes:** Funcionalidade que gamifica o consumo e premia os clientes mais leais.
